📋 Lista Completa — Features & Melhorias do MC

🎨 UI / UX

| # | Feature | Detalhe |
|---|---|---|
| 1 | Agrupar seletores verticalmente | Usar nome ao invés de ícone na criação de task |
| 2 | Legenda da cron expression | Abaixo do código cron, mostrar texto legível em português (ex: "toda semana, seg a sex, às 9h") via cronstrue |
| 3 | ⭐ Favoritar task | Marcar tasks como favoritas para acesso rápido |
| 4 | Comentários no card | Enviar msg na thread sem acionar agente — só contexto para humanos e agentes lerem depois |

---

🤖 Agentes

| # | Feature | Detalhe |
|---|---|---|
| 5 | Conversar diretamente com agente específico | Chat 1:1 com um agente sem criar task |
| 6 | Mencionar agentes via @arroba | Em qualquer task/comentário, acionar agente com @nome |
| 7 | Agente chama outro agente dinamicamente | Configurado na criação da task — agente pode delegar para outro sem passar pelo humano |

🏷️ Tags

| # | Feature | Detalhe |
|---|---|---|
| 9 | Atribuir e editar tags após task criada | Hoje só dá para adicionar na criação |
| 10 | Tags com atributos | Tag vira mini-entidade com campos (cliente, prazo, contrato, prioridade) — funciona como CRM leve |
